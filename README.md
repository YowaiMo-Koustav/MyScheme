# 🚀 AI Government Scheme Matcher

A premium, AI-powered platform that helps users discover the most relevant Indian government schemes based on their profile, needs, and eligibility.

---

## 🌟 Overview

Government schemes are often difficult to search, compare, and understand. This project solves that problem by acting as a **scheme intelligence assistant** rather than a basic search box.

Users enter their details once, and the system returns a ranked list of schemes with:

- ✅ Eligibility match score  
- 🧠 Reasons for matching  
- 📄 Required documents  
- 🛣️ Application steps  
- 🔗 Official source links  
- ⏳ Deadline or status  
- 💾 Save and compare options  

The goal is to make scheme discovery **clear, personalized, trustworthy, and easy to use**.

---

## ✨ Key Features

- 🤖 AI-assisted scheme matching  
- 📏 Rule-based eligibility scoring for transparency  
- 🏆 Ranked recommendations by relevance and fit  
- 💬 Explainable results (“Why you qualify”)  
- 🔗 Official source links and trust indicators  
- 📋 Document checklist for each scheme  
- 💾 Save and compare shortlisted schemes  
- 📱 Mobile-first, accessible UI/UX  
- 🌐 Multilingual-ready structure  
- 🎬 Premium scroll-based landing page  
- 🧾 Detailed scheme pages with application guidance  

---

## 🧭 User Flow

1. 👋 User lands on the cinematic homepage  
2. 🎯 Selects role or life situation  
3. 📝 Fills a short profile form  
4. ⚙️ AI + rules identify matching schemes  
5. 📊 Results are ranked and explained  
6. 🔍 Opens scheme detail page  
7. 💾 Saves, compares, or applies

## 🧠 Matching Logic

The system combines:

- User profile data  
- Eligibility rules  
- Relevance scoring  
- Benefit value  
- Urgency  
- Document readiness  

Each result shows:

- 📊 Match percentage  
- 💬 Explanation  
- ❌ Missing eligibility  
- 📄 Missing documents  
- 🔍 Confidence level  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js + Tailwind CSS  
- **Backend:** FastAPI / Node.js  
- **Database:** PostgreSQL  
- **AI Layer:** LLM + rule-based engine  
- **Search:** Vector database  
- **OCR:** Document extraction
